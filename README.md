# 这是一个常用js集合的小型库
这个库是个前端时代常用得js小集合，我收集整理并修正后放入这个库，这个库会长期更新，更新后会在git中写出，敬请留意（喜欢得朋友不要吝啬自己的小星星哦）  

[文档地址](https://github.com/aiyuekuang/esn/blob/master/wendang.md)  

# 下载方式
    npm install esn

# 使用方式
    import {cutstr} from 'esn';
    
    cutstr('还是大家还是觉得好就是当今世界的技术的',6)
    
    大概就是这样
